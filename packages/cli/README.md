# @spencer-kit/coder-studio

CLI runtime manager for Coder Studio.

## Install

```bash
npm install -g @spencer-kit/coder-studio
```

## Common Commands

```bash
coder-studio start
coder-studio stop
coder-studio restart
coder-studio status
coder-studio logs -f
coder-studio open
coder-studio doctor
coder-studio config show
coder-studio config validate
coder-studio auth status
```
